# HTML5 Bouncing Ball Animation

A browser-based animation created with HTML5 Canvas and JavaScript. The application simulates a bouncing ball that changes to a random color every time it collides with the edge of the canvas.

## Features

- HTML5 Canvas animation
- Smooth ball movement
- Collision detection
- Random color generation
- Automatic bouncing physics
- Browser-based application

## Technologies Used

- HTML5
- CSS
- JavaScript
- Canvas API

## Skills Demonstrated

- HTML5 Canvas
- JavaScript animation
- Collision detection
- Event handling
- Random number generation
- Basic physics simulation
- Browser graphics programming

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Watch the ball bounce around the canvas while changing colors.
